## ⚙️ Configuration

The **Bridger Western Script** ships with a configuration block at the top of the file that you can edit before executing. Every setting has a default value that works for most players, but tweaking these lets you fine-tune behavior for your playstyle.

| Setting | Default | Description |
|---------|---------|-------------|
| `AutoFishEnabled` | `true` | Toggles the auto fishing loop |
| `FarmRadius` | `150` | Maximum distance for auto farm collection |
| `SilentAimFOV` | `90` | Field of view cone for aim assist |
| `SpeedMultiplier` | `1.5` | Walk speed boost factor |
| `AntiAFKInterval` | `120` | Seconds between anti-AFK inputs |
| `AutoSellThreshold` | `80` | Inventory percentage to trigger selling |
| `ESPEnabled` | `false` | Player ESP off by default |
| `TeleportCooldown` | `5` | Seconds between teleport actions |

```bash
-- Bridger Western Script Configuration
-- Edit these values before executing

local Config = {
    AutoFish = {
        Enabled = true,
        BaitType = "Worm",          -- Options: "Worm", "Cricket", "Minnow"
        ReelDelay = 0.3,            -- Seconds to wait before reeling
        CastDistance = "Max",       -- Options: "Short", "Medium", "Max"
    },
    AutoFarm = {
        Enabled = true,
        Radius = 150,               -- Studs around player to farm
        CollectDrops = true,        -- Pick up ground items
        SellWhenFull = true,        -- Auto sell at threshold
    },
    Combat = {
        SilentAim = false,          -- Toggle silent aim
        AimFOV = 90,                -- Field of view in degrees
        NoRecoil = true,            -- Remove weapon recoil
        TargetPriority = "Nearest", -- Options: "Nearest", "Lowest HP"
    },
    Movement = {
        SpeedBoost = 1.5,           -- Multiplier (keep under 2.0)
        TeleportDelay = 5,          -- Cooldown in seconds
    },
    Misc = {
        AntiAFK = true,
        ESPPlayers = false,
        ESPDistance = 500,           -- Max render distance for ESP
        GUIPosition = "TopLeft",    -- Options: "TopLeft", "TopRight", "Center"
    }
}
```

> **Tip:** Save your edited configuration as a separate file at `C:/Users/YourName/Documents/BridgerConfig.lua` and the script will auto-detect it on next launch.

## 🛠️ Troubleshooting Common Issues

**❌ Script does not execute after pasting**
Your executor may not be injected properly into the Roblox process. Close both applications completely, relaunch your executor first, wait for full initialization, then open Roblox and try again.
**Solution:** Restart both programs in the correct order — executor first, then Roblox.

**❌ GUI appears but buttons do not respond**
This usually means an older version of **Bridger Western Script** is cached locally. Clear your executor's script cache folder before loading the latest version.
**Solution:** Delete cached scripts from your executor's `workspace` or `autoexec` directory and re-paste.

**❌ Auto Fish stops working mid-session**
The game occasionally updates fishing mechanics server-side without a client patch. Pull the newest **Bridger Western Script Auto Fish** module from this repository, as hotfixes usually land within hours.
**Solution:** Re-pull the latest code from the **Bridger Western Script Github** page and re-execute.

**❌ Teleport sends you under the map**
Some map coordinates shift when the game developer updates terrain. Check the changelog for coordinate patches.
**Solution:** Update to the latest version where teleport waypoints are corrected.

**❌ Speed Boost causes rubber-banding**
Your multiplier is set too high for your current ping. Reduce `SpeedMultiplier` to `1.2` or `1.3` and test again.
**Solution:** Lower the value in your config and reconnect to a server closer to your region.
